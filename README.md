# 🔐 Day 21 - Custom Password Generator

This is a Python-based password generator that allows users to customize their password based on preferences.

## 🚀 Features
- User-defined password length
- Option to include:
  - Letters (A-Z, a-z)
  - Numbers (0-9)
  - Symbols (!@#$ etc.)
- Input validation to prevent empty character selection

## 🧠 Concepts Used
- input() and type casting
- if conditions
- for loop
- string module
- random module

## ▶️ How to Run

1. Clone the repository
2. Open terminal in project folder
3. Run:

```bash
python main.py
